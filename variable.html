변수 ( variable )

변수의 타입유형

  • 문자형

    [ char ] : 하나의 문자만 저장할 수 있는 공간 :: `A` , `B` , ... 큰따음표를 사용해야하여 저장한다. 아스키코드(2진수를 16진수로)로 변환이 가능하며, 정수형으로 형변환하여 사용할 수 있다.

  • 정수형

    [ byte, short, int ,long ]

    기본형은 int고 사칙연산을 한 결과는 무조건 int로 반환한다. 단 , 소수가 꼭 필요한 경우 별도의 형변환을 통해 소수를 얻을 수 있다.

  • 실수형

    [ float, double ] : 소수

  • 논리형

    [ boolean ]

    true or false의 값을 반환하며, 기본값은 0(false) 이다.

  • 참조형

    [...?]

    기본형 데이터타입 4개 이외 타입이 필요한 경우, 개발자가 직접 만들어서 사용할 수 있는 데이터 타입
